## Setup Instructions

1. Clone the repository:
```bash
git clone [your-repo-url]
cd f1-app
```

2. Open the project in Android Studio (Hedgehog or later)

3. Sync Gradle files

4. Run the app on an emulator or physical device (Android 7.0+ / API 24+)
